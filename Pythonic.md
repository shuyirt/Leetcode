## string

| name    | function                          | usage                            |
| ------- | --------------------------------- | -------------------------------- |
| split   | split string into two             | local, domain = email.split('@') |
| replace | replace a substr with another one | local.replace('.','')            |
| index   | get the index of substr in a str  | email.index('@')                 |
